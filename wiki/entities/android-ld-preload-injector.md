# Android-LD-Preload-Injector

Android Studio–built LD_PRELOAD injector that aimed to load a shared library on any Android version and architecture. Marked dead / no longer working; useful historically for game-security researchers studying Cheat / injection:android preload tradecraft versus ptrace, Virtual Space, or Zygisk paths. (source: wiki/sources/descriptions/reveny__Android-LD-Preload-Injector.md)

Contrasts with attach-and-inject via [[android-ptrace-injector]], no-root Virtual Space inject via [[android-virtual-inject]], and Magisk Zygisk menus such as [[zygisk-imgui-mod-menu]].
