# Android-Ptrace-Injector

C/C++ Android ptrace-based process injector in the Cheat / injection:android lane. Useful for game-security researchers studying attach-and-inject tradecraft (modding / memory analysis) versus container or Zygisk paths. (source: wiki/sources/descriptions/reveny__Android-Ptrace-Injector.md)

Contrasts with no-root Virtual Space inject via [[android-virtual-inject]] and Magisk Zygisk menus such as [[zygisk-imgui-mod-menu]]; complements Android cheat / menu frameworks ([[android-cheat-template]], [[android-modmenu-semijni]]) that assume a different load model.
