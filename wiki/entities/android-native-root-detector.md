# Android-Native-Root-Detector

Kotlin-centered Android tool for detecting root on device. Useful for anti-cheat engineers and defensive researchers studying the Anti Cheat `Detection:Android root` lane and how root-hide / MagiskHide-style bypasses interact with native detection signals. (source: wiki/sources/descriptions/reveny__Android-Native-Root-Detector.md)

Sits alongside Magisk-focused [[magiskdetector]], multi-check environment probes [[detection]], and broader RASP samples such as [[droidshield]] / freeRASP—opposite root frameworks [[magisk]] and [[kernelsu]].
