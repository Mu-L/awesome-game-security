# Android-Native-Import-Hide

C++ library for hiding and retrieving imports in ELF binaries. Centered on hooking/debugging research and useful for anti-cheat engineers studying the Anti Cheat → Compile Time / import-hiding lane on Android native (`.so`) clients. (source: wiki/sources/descriptions/reveny__Android-Native-Import-Hide.md)

Companion to PE IAT obfuscators such as [[alcatraz]] and ELF pack/protect samples ([[elfuck]], [[woody-woodpacker]])—focused on ELF import hide/retrieve rather than full binary packing.
