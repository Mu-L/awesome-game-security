# Android-Virtual-Inject

Android injector that loads code through a Virtual Space container without requiring root permissions. Useful for game-security researchers studying no-root injection:android tradecraft; explicitly not compatible with games protected by anti-cheat. (source: wiki/sources/descriptions/reveny__Android-Virtual-Inject.md)

Contrasts with root/Zygisk inject paths such as [[zygisk-imgui-mod-menu]] (Magisk Zygisk ImGui menu) and complements Android cheat / menu frameworks ([[android-cheat-template]], [[android-modmenu-semijni]]) that assume a different privilege model.
