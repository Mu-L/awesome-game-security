# Android-Library-Remap-Hide

Android library-remap hide sample: remaps a library so it is harder to find in-process. Requires two separate libraries. Useful for game-security researchers and reverse engineers studying offensive cheat / hide tradecraft on Android native (`.so`) clients. (source: wiki/sources/descriptions/reveny__Android-Library-Remap-Hide.md)

Companion to ELF import hide/retrieve via [[android-native-import-hide]]—focused on remapping/hiding a loaded library rather than import-table hide/retrieve.
