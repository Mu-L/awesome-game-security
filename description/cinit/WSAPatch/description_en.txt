This project is a patch for Windows Subsystem for Android (WSA) that enables it to run on unsupported Windows 10 builds and older Windows 11 versions. It modifies WSA binaries to bypass version checks, disable Hyper-V requirement enforcement, and apply compatibility fixes. The C++ patcher operates on WSA MSIX packages to enable Android app sideloading on a wider range of Windows configurations. It is aimed at users and developers wanting to run Android apps on unsupported Windows versions.
