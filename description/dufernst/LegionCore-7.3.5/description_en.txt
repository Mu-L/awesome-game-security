This project is a World of Warcraft server emulator for the Legion expansion (7.3.5 client version) built on the TrinityCore framework. The C++ server implements world simulation, dungeon/raid mechanics, artifact weapons, class halls, world quests, and Legion-specific content. It handles player management, NPC scripting, spell processing, and the WoW network protocol. It is aimed at WoW private server developers and game server researchers studying modern MMO expansion server implementation.
