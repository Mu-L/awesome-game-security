This project is Winlator, an Android application that runs Windows x86/x64 applications and games on ARM-based Android devices. It combines Box86/Box64 for x86-to-ARM binary translation with Wine for Windows API compatibility, and includes a PRoot-based Linux container, Mesa Turnip or VirGL for GPU acceleration, and a virtual desktop environment with touch controls. It is aimed at Android users and developers interested in running Windows software on mobile devices and researchers studying cross-architecture binary translation.
