This project is an Escape from Tarkov cheat that uses a streaming/external approach to display game information outside the game window. It reads EFT's game memory externally through a driver and renders radar, player positions, loot information, and other ESP data on a separate display or overlay. The approach avoids internal injection detection. It is aimed at game security researchers studying external/streaming cheat architectures for Unity-based games with BattlEye protection.
