This project is UnrealCLR, a plugin that embeds the .NET runtime (CoreCLR) into Unreal Engine, enabling game logic development in C#. It provides .NET bindings for UE4/UE5 core systems including actors, components, input, physics, rendering, and the blueprint system. Developers can write gameplay code in C# that interoperates with native C++ and blueprint systems. It is aimed at Unreal Engine developers who prefer C# for game logic while leveraging UE's native rendering and tools.
