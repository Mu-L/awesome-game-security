This project is Tracy, a real-time frame profiler for C++ applications including games. It provides CPU profiling with zone markup, GPU profiling for OpenGL/Vulkan/Direct3D, memory allocation tracking, lock contention analysis, context switch visualization, and source code annotation. The profiler consists of a lightweight client library embedded in the application and a standalone viewer that displays timeline, flame graph, and statistical views. It is aimed at game developers and performance engineers optimizing frame times and identifying bottlenecks in real-time applications.
