This project is ClassyShark, a standalone Android binary inspection tool from Google. It analyzes APK, DEX, AAR, and class files, displaying class hierarchies, method counts, field layouts, dependencies, and multidex structure in an interactive GUI. The Java tool provides quick triage of Android application complexity, third-party SDK usage, and native library dependencies. It is aimed at Android developers and security analysts performing rapid APK structure analysis and dependency auditing.
