This project is Diaphora, the most advanced open-source binary diffing tool, implemented as an IDA Pro plugin in Python. It performs function-level comparison between two binaries using multiple heuristics including control flow graph matching, basic block hash comparison, instruction mnemonics, string references, and call graph topology. It supports partial matching, porting of symbols and comments between database versions, and generates detailed diff reports. It is aimed at reverse engineers, vulnerability researchers, and patch analysts comparing binary updates and identifying code changes across software versions.
