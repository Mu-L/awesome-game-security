This project is a comprehensive catalog of Living Off The Land Drivers (LOLDrivers) - legitimate signed Windows drivers that can be abused for malicious purposes. It includes YAML-based driver metadata, YARA signature enrichment, HVCI driver tracking, Jinja2-based site generation, and validation tools for maintaining the driver database. It is mainly useful for BYOVD researchers, threat hunters, and anti-cheat engineers tracking vulnerable and exploitable signed Windows drivers.
