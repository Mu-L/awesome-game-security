This project is Bullet Physics SDK, an open-source real-time collision detection and multi-physics simulation library written in C++. It provides rigid body dynamics, soft body dynamics, vehicle simulation, character controllers, and constraint solving for games, robotics, visual effects, and scientific computing. The library includes GPU-accelerated broadphase via OpenCL and integrates with major game engines and simulation environments. It is aimed at game developers, robotics researchers, and VFX engineers implementing physics simulation.
