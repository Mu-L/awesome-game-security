GhidraScripts is a maintained collection of scripts for extending and automating analysis workflows inside Ghidra. It is primarily Java-based and includes tooling for AI-assisted renaming, complexity visualization, Golang support, BSim-based matching, and FunctionID database generation. The project also integrates external analysis resources such as SHAREM and Malpedia data to enrich reverse engineering context. It is designed for malware analysts and reverse engineers who need scalable static analysis automation, including workflows relevant to game security investigations.
