This project focuses on this is a research hypervisor written as a UEFI and Windows driver for the educational purpose for Intel processors.
This MiniVisor, as a UEFI driver, provides the ability to inspect system activities even before the operating system boots, while as a Windows driver, allows developers to debug it with familiar tools like WinDbg.
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / efi driver area.
