This project is the Storm Engine, the open-sourced game engine originally developed by Akella for the Sea Dogs/Pirates of the Caribbean series. The C++ engine includes a DirectX 9 renderer, physics system, scripting engine, resource management, scene graph, water simulation, and ship combat mechanics. The codebase represents a complete commercial game engine from the early 2000s era. It is aimed at game engine historians, hobbyist developers, and researchers studying vintage commercial game engine architecture.
