Xiaomi_Kernel_OpenSource is an index-style repository that catalogs many Xiaomi and Redmi kernel source branches by device and Android version.
The content is primarily Markdown tables pointing to branch tags and base references for different hardware platforms and releases.
It serves as a structured map for locating vendor kernel sources needed for driver development, auditing, and platform-specific debugging.
The main audience is Android kernel developers and security researchers working on Xiaomi device internals.
