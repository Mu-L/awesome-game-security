This project is an Android device tree for the Xiaomi Redmi K60 / POCO F5 Pro (mondrian) targeting the Pixel Experience Plus ROM. It includes board configuration, audio and display configs, mixer paths, media codec definitions, and vendor-specific overlay files for the Qualcomm Waipio platform. It is mainly useful for Android ROM developers and mobile security researchers working with custom builds on Xiaomi SM8475-based devices.
