This project is idaxex, an IDA Pro loader plugin for Xbox 360 XEX (Xenon Executable) files. It handles XEX file format parsing, PE extraction from the XEX container, import/export resolution, and Xbox 360 kernel function naming. The C++ plugin enables analysis of Xbox 360 game executables within IDA Pro with proper symbol and structure annotation. It is aimed at reverse engineers analyzing Xbox 360 game binaries and console security researchers.
