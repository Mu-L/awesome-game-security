McDota is a Linux cheat and tinkering framework for Dota 2 focused on runtime injection and in-game tooling.
It is written in C++ and includes features such as dynamic Panorama UI integration, ESP-style visuals, camera controls, and protobuf packet interception and editing.
The repository provides multiple loading modes, including standard, stealthier, and integrated workflows, along with debugging scripts.
It is primarily intended for reverse engineers and game security hobbyists studying Source 2 client behavior on Linux.
