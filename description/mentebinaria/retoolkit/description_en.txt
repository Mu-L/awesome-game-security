Curated Windows installer (built with Inno Setup) that bundles 80+ reverse engineering tools into a single deployment, including debuggers (x64dbg, HyperDbg), decompilers (Ghidra, Cutter), .NET analyzers (dnSpyEx, ILSpy, de4dot), PE inspectors (PE-bear, pestudio, DIE), hex editors (ImHex, HxD), network tools (FakeNet, nmap), and YARA/YARA-X signature scanners.
