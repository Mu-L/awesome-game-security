This project is a collection of dumped Dota 2 game data including netvar offsets, interface pointers, and class structure definitions extracted from the Source 2 engine. It provides C++ headers with entity field offsets, ability data structures, and game state layouts updated after game patches. It is aimed at Dota 2 modders and game security researchers who need current memory layout information for the Source 2 engine as used in Dota 2.
