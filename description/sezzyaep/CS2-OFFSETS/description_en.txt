This project is a regularly updated repository of Counter-Strike 2 memory offsets and netvar dumps. It provides struct offsets, entity field addresses, and interface pointers extracted from the CS2 client and server binaries after each game update. The data is formatted as C++ headers and JSON for use in cheat and mod development. It is aimed at CS2 cheat developers and game security researchers who need current memory layout information for the Source 2 engine.
