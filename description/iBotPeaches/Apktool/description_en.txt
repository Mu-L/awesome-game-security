This project is Apktool, a tool for reverse engineering Android APK files. It decodes resources to nearly original form, including layouts, strings, and assets, and rebuilds them after modification. It supports smali disassembly/reassembly of DEX bytecode, resource table decoding, AndroidManifest.xml parsing, and APK repackaging with proper resource compilation. The Java tool enables APK modification without access to original source code. It is aimed at Android reverse engineers, modders, and security researchers who need to inspect, modify, and repackage Android applications.
