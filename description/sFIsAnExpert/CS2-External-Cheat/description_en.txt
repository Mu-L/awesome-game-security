This project is an external cheat for Counter-Strike 2 with a D3D11 ImGui overlay interface. The C++ codebase includes aimbot, engine interaction, math utilities, memory reading modules, offset management, and a DirectX 11-based overlay renderer for displaying cheat UI and ESP visuals. It is mainly useful for game security researchers studying external cheat architecture, D3D11 overlay rendering, and CS2 memory reading techniques.
