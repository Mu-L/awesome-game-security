This project is a Windows desktop tool for inspecting and working with Nintendo Switch XCI and NSP package contents. It is written in C# on .NET WinForms and includes parsers for container formats such as XCI, NCA, HFS0, and PFS0. Key features include metadata viewing, partition browsing, hash checking, file extraction, and certificate editing workflows. It is mainly used by Switch modding and game security researchers who need to analyze cartridge images and package structure.
