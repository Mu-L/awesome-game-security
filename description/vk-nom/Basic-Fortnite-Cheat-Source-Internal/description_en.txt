This project is a basic Fortnite internal cheat source in C++ that demonstrates core cheat functionality for Unreal Engine 4-based games. It includes player ESP, aimbot, and basic visual modifications implemented through UE4 SDK interaction, GObject/GNames enumeration, and engine function hooking. The codebase shows the typical structure of an internal UE4 game cheat with process-wide hooks. It is aimed at game security researchers studying Fortnite cheat architecture and UE4-based cheat detection patterns.
