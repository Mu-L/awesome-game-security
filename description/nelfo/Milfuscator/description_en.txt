This project is the 'Free Milfuscator' is now an open-source project, the essence of which is to mutate the x32 code by rebuilding the entire PE using the Zydis & AsmJit libraries.
The very idea of mutation is taken from one P2C project for CS:GO.
It is mainly useful for anti-cheat engineers and defensive security researchers working in the anti cheat / obfuscation engine area.
