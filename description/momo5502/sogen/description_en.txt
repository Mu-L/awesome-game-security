Windows userspace emulator built on Unicorn Engine and Capstone that executes x86/x64 PE binaries with full syscall emulation, minidump loading, and Zstd-compressed state serialization. Includes a web-based UI (React/TypeScript) for interactive emulation sessions, cross-compilation support via Emscripten/Android NDK, and FlatBuffers-based IPC for structured execution traces.
