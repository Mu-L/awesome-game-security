This project is a reverse-engineered internal SDK for Counter-Strike 2 containing Source 2 engine class definitions, interface pointers, netvar offsets, and schema system structures. The C++ headers provide type-accurate representations of CS2's entity system, player classes, weapon data, and rendering structures for internal cheat development. It is aimed at CS2 cheat developers and game security researchers who need an accurate Source 2 SDK for hooking and memory access.
