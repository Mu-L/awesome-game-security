This project is an internal cheat for Rainbow Six Siege written in C++ that hooks into the game process to provide ESP, aimbot, and player information features. It reads game entity data through direct memory access using reverse-engineered SDK structures and renders overlays through the game's own rendering pipeline. The codebase demonstrates Unreal Engine-based FPS cheat patterns specific to R6S. It is aimed at game security researchers studying Rainbow Six Siege cheat implementations and anti-cheat detection vectors.
