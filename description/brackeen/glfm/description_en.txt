This project is a C API for mobile app development with OpenGL ES and Metal, inspired by GLFW.
It supports iOS 9+, tvOS 9+, Android 4.1+, and WebGL via Emscripten, with features including OpenGL ES 2/3 and Metal display setup, retina/high-DPI support, touch and keyboard events, and sensor input (accelerometer, gyroscope, magnetometer).
The single-header library provides cross-platform windowing, input handling, and application lifecycle management for mobile and web platforms.
It is mainly useful for mobile game developers and cross-platform graphics programmers building OpenGL ES or Metal applications.
