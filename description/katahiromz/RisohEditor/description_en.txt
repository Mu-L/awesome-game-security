This project is RisohEditor, a free Windows resource editor for PE executable and DLL files. It provides a GUI for viewing, editing, adding, and removing resources including dialogs, menus, icons, bitmaps, string tables, version info, and manifest files. The C++ application supports resource script (RC) import/export and handles both 32-bit and 64-bit PE files. It is aimed at Windows developers, reverse engineers, and translators who need to modify embedded resources in compiled executables.
