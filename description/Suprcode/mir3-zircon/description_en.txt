This project is an open-source engine and tooling stack for The Legend of Mir 3. It is primarily written in C# and contains client, server, launcher, editor, and shared library projects for a complete MMORPG workflow. The codebase includes networking layers, content/data models, and rendering components based on DirectX/SharpDX pipelines. It is intended for Mir server operators and game development communities building custom content, server logic, and modernized private deployments.
