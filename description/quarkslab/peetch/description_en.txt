This project focuses on eBPF toolkit: dump (sniff traffic with PID/process), tls (OpenSSL key/master-secret extraction), proxy (intercept and decrypt TLS); PCAPng + Scapy; OpenSSL, IPv4, TLS 1.2.
The first called dump aims to sniff network traffic by associating information about the source process with each packet.
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / android kernel explorer area.
