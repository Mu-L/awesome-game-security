This project is a sample application demonstrating the Windows Desktop Duplication API (DXGI Output Duplication) for capturing the desktop screen content. It shows how to acquire frames from the desktop compositor, access pixel data, handle cursor rendering, and detect dirty/moved regions for efficient screen capture. The C++ sample uses DirectX 11 and DXGI 1.2. It is aimed at developers building screen capture tools, remote desktop applications, or screenshot-based anti-cheat systems.
