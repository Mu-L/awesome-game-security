This project is an extension for dnSpyEx that assists with reversing obfuscated .NET binaries.
It is developed in C# and adds capabilities such as source-map-style renaming, managed DLL injection during debugging, and native disassembly support.
The extension also provides control-flow graph visualization and utility commands for extension development.
It is useful for reverse engineers, malware analysts, and game modders working with protected managed code.
