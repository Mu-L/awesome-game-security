Kotek は、三層ゲームエンジンスタックの基盤層をなすモジュラーな C/C++ コアフレームワークであり、上位のエンジンおよびゲームコードが依拠できる差し替え可能なサービスを提供します。多数の小さな CMake モジュールとして構成され、安定した ktkI* インターフェースがメインマネージャー経由で登録されるため、数学、コンテナ、ログ、ファイルシステム、ウィンドウ、UI、ECS、レンダリングなどの各バックエンドを構成時に切り替えたり、プラグインで実行時に上書きしたりできます。対応バックエンドには GLM または DirectXMath、GLFW、Dear ImGui、RmlUi、CEF、pico_ecs、bgfx などの選択肢があり、実用的な範囲では依存なしの独自実装も用意されています。ビルドは Windows を第一対象とし、Linux の OS 抽象も備え、静的・共有・明示的プラグインリンクの各モードをサポートします。主な用途は、カスタマイズ可能で依存を切り替えられる C++ ランタイムコアを必要とするゲームおよびエンジン開発です。
