Kotek 是一個模組化的 C/C++ 核心框架，構成三層遊戲引擎堆疊的基礎層，為上層引擎與遊戲程式碼提供可替換的服務。它由眾多小型 CMake 模組組成，透過主管理器註冊穩定的 ktkI* 介面，因此數學、容器、日誌、檔案系統、視窗、UI、ECS 與渲染等各個後端可在設定時切換，或在執行時透過外掛覆寫。支援的後端包括 GLM 或 DirectXMath、GLFW、Dear ImGui、RmlUi、CEF、pico_ecs 與 bgfx 等選項，並在可行時提供無相依的自實作。建置以 Windows 為優先目標，同時具備 Linux 作業系統抽象，並支援靜態、共享與明確外掛連結模式。主要用途是需要可自訂、可切換相依的 C++ 執行時期核心的遊戲與引擎開發。
