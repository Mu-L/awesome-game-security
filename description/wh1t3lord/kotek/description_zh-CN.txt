Kotek 是一个模块化的 C/C++ 核心框架，构成三层游戏引擎栈的基础层，为上层引擎与游戏代码提供可替换的服务。它由众多小型 CMake 模块组成，通过主管理器注册稳定的 ktkI* 接口，因此数学、容器、日志、文件系统、窗口、UI、ECS 与渲染等各个后端可在配置时切换，或在运行时通过插件覆盖。支持的后端包括 GLM 或 DirectXMath、GLFW、Dear ImGui、RmlUi、CEF、pico_ecs 与 bgfx 等选项，并在可行时提供无依赖的自实现。构建以 Windows 为优先目标，同时具备 Linux 操作系统抽象，并支持静态、共享与显式插件链接模式。主要用途是需要可定制、可切换依赖的 C++ 运行时核心的游戏与引擎开发。
