Kotek es un framework núcleo modular en C/C++ que forma la capa base de una pila de motor de juego de tres capas, proporcionando servicios reemplazables sobre los que puede construirse el código de motor y de juego de niveles superiores. Se organiza en muchos módulos CMake pequeños con interfaces ktkI* estables registradas a través de un gestor principal, de modo que los backends individuales de matemáticas, contenedores, registro, sistema de archivos, ventanas, UI, ECS y renderizado pueden intercambiarse en tiempo de configuración o sobrescribirse en tiempo de ejecución mediante plugins. Los backends admitidos incluyen opciones como GLM o DirectXMath, GLFW, Dear ImGui, RmlUi, CEF, pico_ecs y bgfx, junto con implementaciones propias sin dependencias cuando resulta práctico. Las compilaciones priorizan Windows con abstracciones de SO para Linux, y admiten modos de enlace estático, compartido y de plugin explícito. El caso de uso principal es el desarrollo de juegos y motores donde se necesita un núcleo de runtime C++ personalizable y con dependencias intercambiables.
