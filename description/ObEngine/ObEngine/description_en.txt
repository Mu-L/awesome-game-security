ObEngine is an open-source 2D game engine focused on accessible game development and Lua-driven scripting. It is primarily written in C++ and bundles systems such as scene and game object management, map tooling, animation, input binding, networking, and plugin support. The engine integrates SFML and other bundled dependencies while exposing runtime features through scriptable APIs and project tooling. It targets indie developers and learners building 2D games with customizable engine internals.
