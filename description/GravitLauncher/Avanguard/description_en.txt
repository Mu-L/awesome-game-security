Avanguard is a Win32 anti-intrusion library for hardening x86 and x64 applications against tampering. The C and C++ codebase includes anti-debugging techniques, anti-injection controls, memory and call stack analysis, integrity checks, and protection utilities. It also supports process-level defenses such as callback filtering, module checks, and related defensive APIs with optional bindings. Its primary use case is protecting game clients and other sensitive software from debugging, injection, and runtime manipulation.
