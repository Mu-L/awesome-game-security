This project is OpenProcmon, an open-source process monitor for Windows inspired by Sysinternals Process Monitor. It captures real-time system activity including process creation/termination, file system operations, registry access, network activity, and DLL loading using ETW (Event Tracing for Windows) and minifilter drivers. The C++ application provides a GUI with filtering, highlighting, and export capabilities. It is aimed at system administrators, malware analysts, and security researchers monitoring Windows system activity.
