This project is tinyobjloader, a lightweight single-header C++ library for parsing Wavefront OBJ 3D model files. It loads vertex positions, normals, texture coordinates, materials (MTL files), and face indices with support for triangulation, smoothing groups, and per-face materials. The header-only implementation has no external dependencies and handles large models efficiently. It is aimed at game developers and graphics programmers who need a simple, portable OBJ file loader for 3D rendering applications.
