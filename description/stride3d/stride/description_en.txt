This project is Stride (formerly Xenko), an open-source C# game engine with a full-featured editor. It provides PBR rendering, global illumination, a visual scripting system, physics (Bullet), audio, UI, animation, particle systems, and VR support. The engine uses a modern rendering pipeline with Vulkan, Direct3D 11/12, and OpenGL backends. It is aimed at game developers building 3D games and interactive applications using C# and .NET.
