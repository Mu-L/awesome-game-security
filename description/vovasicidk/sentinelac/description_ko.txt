Project Sentinel는 얇은 프로세스 내 SDK, 별도의 서명된 유저모드 서비스, 커널모드 드라이버로 게임 클라이언트를 보호하는 Windows 우선 안티치트 스택입니다. SDK는 작은 Init/Heartbeat/Shutdown API를 제공하고, 수동 오프셋 없이 PE .text와 import를 자동 매핑해 무결성 해싱을 수행하며, named pipe 또는 ALPC로 서비스와 통신해 집행이 게임 바이너리 밖에 남도록 합니다. 커널 에이전트는 ObRegisterCallbacks, 로드 이미지 알림 루틴 등 문서화된 콜백으로 프로세스 보호와 무단 드라이버 탐지를 수행하고, 유저모드 구성 요소는 콜 스택 검증과 오버레이·윈도우 스테이션 가드를 더합니다. Node.js 백엔드는 mTLS로 텔레메트리를 수집하고, 서명(예: Postgres)을 저장하며, SSE로 블랙리스트 업데이트를 전파해 분석가 주도 탐지·대응을 지원합니다. 완성된 상용 제품이라기보다 클라이언트 안티치트, 무결성 모니터링, 치트 서명 배포를 구축하는 게임 스튜디오를 대상으로 합니다.
