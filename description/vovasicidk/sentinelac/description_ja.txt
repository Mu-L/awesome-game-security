Project Sentinel は、薄いプロセス内 SDK、別途署名されたユーザーモードサービス、およびカーネルモードドライバーによりゲームクライアントを保護する Windows 優先のアンチチートスタックです。SDK は小さな Init/Heartbeat/Shutdown API を公開し、手動オフセットなしで PE の .text とインポートを自動マップして整合性ハッシュを行い、名前付きパイプまたは ALPC 経由でサービスと通信するため、執行はゲームバイナリの外に保たれます。カーネルエージェントは ObRegisterCallbacks やロードイメージ通知ルーチンなどのドキュメント化されたコールバックを用いてプロセス保護と不正ドライバー検出を行い、ユーザーモード側ではコールスタック検証やオーバーレイ／ウィンドウステーションのガードを追加します。Node.js バックエンドは mTLS 経由でテレメトリを取り込み、署名を（例えば Postgres に）保存し、SSE でブラックリスト更新を配信して、アナリスト主導の検知と対応を支えます。完成された商用製品というより、クライアントアンチチート、整合性監視、チート署名配信を構築するゲームスタジオ向けです。
