Project Sentinel 是一套以 Windows 為主的反作弊技術堆疊，透過精簡的程序內 SDK、獨立的已簽章使用者模式服務以及核心模式驅動程式來保護遊戲用戶端。SDK 提供簡潔的 Init/Heartbeat/Shutdown API，可自動對應 PE 的 .text 區段與匯入表以進行完整性雜湊而無需手動偏移，並透過命名管道或 ALPC 與服務通訊，使執法邏輯留在遊戲二進位之外。核心代理使用 ObRegisterCallbacks、映像載入通知等已文件化的回呼實作程序保護與未授權驅動程式偵測，使用者模式元件則補充呼叫堆疊驗證以及覆蓋層或視窗站防護。Node.js 後端透過 mTLS 接收遙測資料，將簽章儲存（例如在 Postgres 中），並透過 SSE 分發黑名單更新，以支援分析人員驅動的偵測與回應。它面向需要建置用戶端反作弊、完整性監控與作弊簽章分發的遊戲工作室，而非作為成品商業產品。
