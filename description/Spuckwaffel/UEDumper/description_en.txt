This project is an all-in-one Unreal Engine dumper and live memory editor.
It supports Unreal Engine versions from 4.19 through 5.3.0 and includes SDK generation, structure dumping, and runtime browsing utilities.
The implementation is C++ with an ImGui-based interface and components for object management, type generation, and engine-specific handling.
It is widely useful for Unreal game reverse engineering and game security research workflows.
