This project is ArkDecompiler, a decompiler for Huawei's ArkCompiler bytecode format used in HarmonyOS applications. It parses ArkTS/ArkJS compiled bytecode, reconstructs control flow, recovers source-level constructs, and outputs readable JavaScript/TypeScript code from compiled .abc files. It is aimed at mobile security researchers and reverse engineers analyzing HarmonyOS applications built with ArkCompiler.
