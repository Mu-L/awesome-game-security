This project is Osiris, an open-source CS:GO cheat written in modern C++. It provides features including ESP (player boxes, health, names), glow outlines, aimbot, triggerbot, backtrack, skin changer, and inventory manipulation. The codebase hooks into the Source engine through interface pointers, pattern scanning, and VMT hooking, demonstrating the internal structure of a feature-complete game cheat. It is aimed at game security researchers studying Source engine cheat architecture and anti-cheat developers building detection for these techniques.
