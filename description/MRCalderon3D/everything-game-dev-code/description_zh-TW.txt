Everything Game Dev Code 是一套面向 AI 輔助遊戲開發的分層工作流腳手架，本身並不是可玩的遊戲。它協調規則、專用代理、斜線式命令、可重用技能、階段上下文與鉤子，使編碼助手能在統一標準下處理設計、實作、QA、發布與 live-ops 工作。該腳手架支援 Unity、Unreal、Godot 以及 HTML/JS（2D 與 3D），並嚴格隔離引擎，同時提供 Claude Code、Codex、Cursor、OpenCode 和 Kiro 的 harness 適配器。Node.js 工具層可驗證並同步清單、結構產物與包裝器；在提供商金鑰可用時，可選的 AI 資產生成路徑還能升級佔位資源。其主要用途是為 AI 驅動的遊戲專案標準化多引擎工作室與個人工作流，包括安全、效能與主機合規等專項審查角色。
