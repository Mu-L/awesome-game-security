Everything Game Dev Code — это многоуровневый каркас рабочих процессов для разработки игр с помощью ИИ, а не сама по себе играбельная игра. Он координирует правила, специализированных агентов, slash-команды, переиспользуемые навыки, контексты фаз и хуки, чтобы ассистенты по коду могли вести дизайн, реализацию, QA, релиз и live-ops по общим стандартам. Каркас поддерживает Unity, Unreal, Godot и HTML/JS (2D и 3D) со строгой изоляцией движков, а также адаптеры harness для Claude Code, Codex, Cursor, OpenCode и Kiro. Слой инструментов на Node.js проверяет и синхронизирует манифесты, структурные артефакты и обёртки; при наличии ключа провайдера опциональный путь ИИ-генерации ассетов может заменить заполнители. Основной сценарий использования — стандартизация мультидвижковых студийных и соло-процессов для ИИ-ориентированных игровых проектов, включая специализированные роли ревью по безопасности, производительности и консольному соответствию.
