Everything Game Dev Code è un’impalcatura di workflow a livelli per lo sviluppo di giochi assistito dall’IA, non un gioco giocabile di per sé. Coordina regole, agenti specializzati, comandi in stile slash, skill riutilizzabili, contesti di fase e hook affinché gli assistenti di codice possano gestire design, implementazione, QA, rilascio e live-ops secondo standard condivisi. L’impalcatura supporta Unity, Unreal, Godot e HTML/JS (2D e 3D) con isolamento rigoroso dei motori, oltre ad adattatori harness per Claude Code, Codex, Cursor, OpenCode e Kiro. Uno strato di tooling Node.js convalida e sincronizza manifest, artefatti di struttura e wrapper; un percorso opzionale di generazione asset con IA può aggiornare i placeholder quando è presente una chiave del provider. Il caso d’uso principale è standardizzare i workflow multi-engine di studio e individuali per progetti di giochi guidati dall’IA, inclusi ruoli di revisione specializzati come security, performance e conformità console.
