Everything Game Dev Code is a layered workflow scaffold for AI-assisted game development, not a playable game itself. It coordinates rules, specialized agents, slash-style commands, reusable skills, phase contexts, and hooks so coding assistants can handle design, implementation, QA, release, and live-ops work under shared standards. The scaffold supports Unity, Unreal, Godot, and HTML/JS (2D and 3D) with strict engine isolation, plus harness adapters for Claude Code, Codex, Cursor, OpenCode, and Kiro. A Node.js tooling layer validates and syncs manifests, structure artifacts, and wrappers, and an optional AI asset-generation path can upgrade placeholders when a provider key is present. Its primary use case is standardizing multi-engine studio and solo workflows for AI-driven game projects, including specialized review roles such as security, performance, and console compliance.
