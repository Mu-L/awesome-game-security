Everything Game Dev Code は、AI 支援のゲーム開発向けの階層型ワークフロー・スキャフォールドであり、プレイ可能なゲーム自体ではありません。ルール、専用エージェント、スラッシュ形式のコマンド、再利用可能なスキル、フェーズコンテキスト、フックを連携させ、コーディングアシスタントが共通の基準のもとで設計、実装、QA、リリース、ライブオペレーションの作業を扱えるようにします。Unity、Unreal、Godot、HTML/JS（2D および 3D）を厳密なエンジン分離とともにサポートし、Claude Code、Codex、Cursor、OpenCode、Kiro 向けのハーネスアダプターも備えます。Node.js ツール層がマニフェスト、構造アーティファクト、ラッパーを検証・同期し、プロバイダーキーがある場合はオプションの AI アセット生成パスでプレースホルダーを置き換えられます。主な用途は、セキュリティ、パフォーマンス、コンソール準拠などの専門レビュー役を含め、AI 駆動のゲームプロジェクト向けにマルチエンジンのスタジオおよび個人ワークフローを標準化することです。
