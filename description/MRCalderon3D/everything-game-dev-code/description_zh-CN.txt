Everything Game Dev Code 是一套面向 AI 辅助游戏开发的分层工作流脚手架，本身并不是可玩的游戏。它协调规则、专用代理、斜杠式命令、可复用技能、阶段上下文和钩子，使编码助手能在统一标准下处理设计、实现、QA、发布与 live-ops 工作。该脚手架支持 Unity、Unreal、Godot 以及 HTML/JS（2D 与 3D），并严格隔离引擎，同时提供 Claude Code、Codex、Cursor、OpenCode 和 Kiro 的 harness 适配器。Node.js 工具层可校验并同步清单、结构产物与包装器；在提供商密钥可用时，可选的 AI 资产生成路径还能升级占位资源。其主要用途是为 AI 驱动的游戏项目标准化多引擎工作室与个人工作流，包括安全、性能与主机合规等专项审查角色。
