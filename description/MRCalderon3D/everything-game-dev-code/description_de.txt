Everything Game Dev Code ist ein geschichtetes Workflow-Scaffold für KI-gestützte Spieleentwicklung, kein spielbares Spiel. Es koordiniert Regeln, spezialisierte Agenten, Slash-Befehle, wiederverwendbare Skills, Phasenkontexte und Hooks, damit Coding-Assistenten Design, Implementierung, QA, Release und Live-Ops unter gemeinsamen Standards übernehmen können. Das Scaffold unterstützt Unity, Unreal, Godot und HTML/JS (2D und 3D) mit strikter Engine-Isolation sowie Harness-Adapter für Claude Code, Codex, Cursor, OpenCode und Kiro. Eine Node.js-Tooling-Schicht validiert und synchronisiert Manifeste, Strukturartefakte und Wrapper; ein optionaler KI-Asset-Generierungspfad kann Platzhalter ersetzen, wenn ein Provider-Schlüssel vorhanden ist. Der Hauptanwendungsfall ist die Standardisierung von Multi-Engine-Workflows für Studios und Einzelpersonen bei KI-getriebenen Spieleprojekten, einschließlich spezialisierter Review-Rollen wie Security, Performance und Console Compliance.
