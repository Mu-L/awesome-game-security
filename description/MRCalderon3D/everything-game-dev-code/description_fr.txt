Everything Game Dev Code est un échafaudage de flux de travail en couches pour le développement de jeux assisté par l’IA, et non un jeu jouable en soi. Il coordonne règles, agents spécialisés, commandes de type slash, compétences réutilisables, contextes de phase et hooks afin que les assistants de code puissent gérer conception, implémentation, QA, publication et live-ops selon des standards partagés. L’échafaudage prend en charge Unity, Unreal, Godot et HTML/JS (2D et 3D) avec une isolation stricte des moteurs, plus des adaptateurs de harness pour Claude Code, Codex, Cursor, OpenCode et Kiro. Une couche d’outils Node.js valide et synchronise les manifests, les artefacts de structure et les wrappers ; un chemin optionnel de génération d’assets par IA peut remplacer les placeholders lorsqu’une clé fournisseur est présente. Son cas d’usage principal est de standardiser les flux de travail multi-moteurs en studio et en solo pour les projets de jeux pilotés par l’IA, y compris des rôles de revue spécialisés tels que sécurité, performance et conformité console.
