This project focuses on duckDetector refactor: Android local device-integrity inspector for root tampering, runtime hooking, mount anomalies, attestation trust, and virtualization evidence.
The project combines a Jetpack Compose UI, modular Kotlin feature packages, and native C++ / assembly probes to surface detector cards with structured findings, method coverage, and scan-state summaries.
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / android root area.
