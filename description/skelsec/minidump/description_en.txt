This project is a Python library for parsing Windows minidump (.dmp) files. It extracts process memory contents, thread contexts, module lists, exception records, and system information from minidump files without requiring Windows or WinDbg. The library supports both full and mini dump formats and can be used programmatically for automated analysis or credential extraction from LSASS dumps. It is aimed at incident responders, forensic analysts, and penetration testers working with Windows memory dumps in cross-platform environments.
