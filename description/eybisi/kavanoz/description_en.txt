This project is Kavanoz, an automated Android DEX and resource unpacker written in Python. It detects and extracts packed or encrypted DEX files from Android applications protected by commercial packers such as Bangcle, Ijiami, Qihoo 360, and others. The tool identifies the packing scheme, locates the encrypted payload, and applies the appropriate decryption or decompression routine. It is aimed at Android malware analysts and reverse engineers who need to unpack protected APKs for static analysis.
