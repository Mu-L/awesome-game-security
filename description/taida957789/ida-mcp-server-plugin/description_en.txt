This project is an IDA Pro plugin that implements a Model Context Protocol (MCP) server, exposing IDA's analysis capabilities to AI assistants and external tools. It allows MCP clients to query disassembly, decompilation, cross-references, function information, and type data from IDA databases remotely. The Python plugin bridges IDA Pro with AI-powered reverse engineering workflows. It is aimed at reverse engineers integrating IDA Pro with LLM-based analysis tools through the MCP protocol.
