This project focuses on benowin Blanc — parse Windows SDK/PHNT via libclang; struct layouts, enums, constants like dt without WinDbg; CLI + TUI, JSON export.
Separately, the community-maintained PHNT (Process Hacker NT headers) documents internal structures that Microsoft doesn't publish.
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / windows kernel explorer area.
