This project is TinyGLTF, a header-only C++ library for loading and saving glTF 2.0 3D model files. It parses both glTF JSON and GLB binary formats, handling meshes, materials, textures, animations, skins, cameras, and extensions without external dependencies beyond stb_image for texture loading. The library supports reading from files, memory buffers, and custom I/O callbacks. It is aimed at game developers and 3D application programmers who need a lightweight, portable glTF loader.
