This project is the Intel Processor Trace (PT) decoder library (libipt), an open-source reference implementation for decoding Intel PT trace packets. It reconstructs instruction-level execution traces from raw PT data, handling timing, control flow, and packet decoding across multiple trace formats. The C library provides APIs for packet-level and instruction-level trace decoding. It is aimed at debugger authors, performance tool developers, and security researchers building Intel PT-based tracing and coverage tools.
