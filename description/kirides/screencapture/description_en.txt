This project is a screen capture library for Windows that provides efficient desktop frame acquisition using the DXGI Desktop Duplication API. It captures the desktop display with minimal CPU overhead by leveraging GPU-based frame copying, handles multi-monitor setups, and outputs frames in standard pixel formats for processing. The C++/C# library is suitable for building screen recording, streaming, or screenshot tools. It is aimed at developers building screen capture functionality for applications or anti-cheat screenshot systems.
