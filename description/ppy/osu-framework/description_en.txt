This project is osu!framework, a C# game framework built for osu! but usable as a general-purpose 2D game engine. It provides a scene graph with drawable hierarchies, input handling, audio management, threading model, smooth interpolation, text rendering, texture atlasing, and platform abstraction over OpenGL. The framework handles game loop timing, resource management, and UI layout. It is aimed at C# game developers wanting a well-tested 2D framework with modern .NET features.
