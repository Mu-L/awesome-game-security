This project is osu!, an open-source rhythm game written in C# using the osu!framework. It implements multiple game modes (osu!standard, osu!taiko, osu!catch, osu!mania) with beatmap parsing, hit object rendering, scoring, replay recording, online multiplayer, and skinning support. The codebase includes a custom UI framework, audio processing pipeline, input handling, and smooth interpolation systems. It is aimed at rhythm game developers, C# game programmers, and the osu! modding community.
