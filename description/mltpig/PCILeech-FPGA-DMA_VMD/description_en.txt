PCILeech FPGA firmware for Xilinx Artix-7 XC7A75T that emulates an Intel RST VMD (Volume Management Device) controller with Device ID 9A0B for DMA-based memory access. Implements multi-function PCI device simulation, MSI-X interrupt handling, NVMe command processing, configurable BAR addressing, and advanced TLP-level PCIe config space shadowing in SystemVerilog.
