Cross-platform C++ 3D game framework providing an OpenGL ES 2.0/3.0 rendering pipeline, scene graph with material/shader system, Bullet physics integration, OpenAL audio, skeletal animation, terrain rendering, particle effects, and a form-based UI toolkit. It targets Windows, macOS, Linux, iOS, and Android from a single CMake-based codebase, with Lua scripting support and an encoder tool for converting FBX/Collada assets into the engine's binary .gpb format.
