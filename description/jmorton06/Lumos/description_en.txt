This project is Lumos, an open-source C++ game engine with a cross-platform editor. It provides Vulkan/OpenGL rendering with PBR materials, a scene graph with ECS architecture, physics (Box2D/Bullet), audio, Lua scripting, 2D/3D support, and a Dear ImGui-based editor interface. The engine includes asset pipelines, serialization, and multi-platform build support. It is aimed at indie game developers and engine enthusiasts building 2D/3D games with an accessible open-source engine.
