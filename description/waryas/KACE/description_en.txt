This project is KACE (Kernel Automated Compatibility Emulator), a Windows kernel emulator that runs kernel-mode drivers in user-mode for analysis. It emulates essential kernel APIs, memory management, IRQL handling, and driver entry point calling conventions, allowing security researchers to load and analyze kernel drivers without risking system stability. The C++ emulator provides a sandboxed environment for studying driver behavior. It is aimed at reverse engineers and anti-cheat analysts who need to safely analyze kernel drivers and anti-cheat components.
