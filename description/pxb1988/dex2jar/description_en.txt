This project is dex2jar, a set of Java tools for converting Android DEX bytecode to Java class files (JAR format) and vice versa. The converted JAR files can then be analyzed with standard Java decompilers like JD-GUI or CFR. The toolset also includes d2j-baksmali for disassembly and utilities for APK signing and DEX manipulation. It is aimed at Android reverse engineers and security analysts who need to examine Android application logic using familiar Java analysis tools.
