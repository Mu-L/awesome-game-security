This project is a full-source modern 3D game engine and editor designed for high-performance game development. It uses a mixed C++ and C# architecture, with rendering and platform layers plus managed scripting and tooling workflows. The repository includes engine runtime code, editor code, build scripts, and cross-platform support for Windows, Linux, and macOS with technologies like .NET and Vulkan. Its primary audience is game developers and engine programmers who need deep source-level control and extensibility.
