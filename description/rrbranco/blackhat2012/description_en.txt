This project is the samples are divided in four categories: Anti-Debugging Anti-Disassembly Obfuscation Anti-VM.
Anti-Debugging POCs were developed in C/C++ using Visual Studio 2010 and Assembly with Flat Assembler, they are in the folder Csrc and ASMsrc, respectively.
It is mainly useful for anti-cheat engineers and defensive security researchers working in the anti cheat / anti disassembly area.
