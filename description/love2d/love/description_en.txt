This project is LOVE (Love2D), a free open-source 2D game framework written in C++ with Lua scripting. It provides modules for graphics rendering (OpenGL/Metal), audio playback, physics (Box2D), windowing, input handling, threading, and filesystem access through a simple Lua API. Games are distributed as .love archive files containing Lua scripts and assets. It is aimed at indie game developers and hobbyists prototyping and building 2D games with minimal boilerplate.
