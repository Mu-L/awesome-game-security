This project is an external CS:GO cheat written in C++ that operates by reading game memory from a separate process. It provides ESP, aimbot, radar hack, and basic visual features through external memory reading using ReadProcessMemory or a kernel driver. The cheat reads entity positions, player health, and weapon data from the game's memory without injecting code into the CS:GO process. It is aimed at game security researchers studying external cheat patterns and anti-cheat external detection methods.
