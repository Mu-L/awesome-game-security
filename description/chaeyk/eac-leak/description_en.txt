This project is a leaked EasyAntiCheat server implementation that interfaces with Epic Online Services (EOS) for anti-cheat session management.
The C++ codebase includes a server component with EOS SDK integration for anti-cheat session handling, requiring EOS application ID replacement for deployment.
It provides insight into the server-side communication protocol between game servers and EAC's cloud validation infrastructure.
It is mainly useful for anti-cheat researchers studying EasyAntiCheat's server-side architecture and EOS-based anti-cheat session management.
