This project is the Eclipse Paho MQTT C++ client library, providing a modern C++ API for MQTT messaging protocol communication. It supports MQTT v3.1, v3.1.1, and v5.0 with features including QoS levels 0-2, persistent sessions, TLS encryption, automatic reconnection, and asynchronous message delivery. The library wraps the Paho C client with RAII-style resource management and callback-based event handling. It is aimed at IoT developers and systems programmers integrating MQTT pub/sub messaging into C++ applications.
