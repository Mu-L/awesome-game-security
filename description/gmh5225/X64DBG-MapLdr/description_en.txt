This project is an x64dbg plugin that loads MAP files (linker map output) to apply symbol names and addresses to the debugged binary. It parses MAP files generated by MSVC, Borland, or other linkers and imports function names, global variable names, and segment information into x64dbg's symbol database. The C++ plugin bridges compile-time symbol information with runtime debugging. It is aimed at reverse engineers using x64dbg who have access to MAP files for the target binary.
