This project is Bytecode Viewer, a Java/Android APK reverse engineering suite that integrates multiple decompilers (CFR, Procyon, FernFlower, JD-GUI, Krakatau) into a single tabbed interface. It supports viewing Java bytecode, decompiled source, and Smali simultaneously for JAR, class, DEX, and APK files. The tool includes search, string extraction, and plugin support. It is aimed at Java and Android reverse engineers who want to compare decompiler outputs and analyze bytecode in a unified environment.
