This project is a Valorant cheat GUI framework using Dear ImGui as the menu rendering system. It provides a styled ImGui interface with tabs, toggles, and sliders for controlling cheat features like ESP, aimbot, and visual modifications in Valorant. The C++ codebase demonstrates cheat menu design and UE4 game interaction patterns. It is aimed at game security researchers studying Valorant cheat UI implementations and ImGui-based cheat menu architectures.
