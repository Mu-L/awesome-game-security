This project documents or exploits miHoYo's mhyprot2.sys anti-cheat kernel driver used in Genshin Impact. The signed driver's IOCTL interface provides kernel read/write and process termination capabilities that can be abused as a BYOVD primitive. This tool demonstrates leveraging mhyprot2 for kernel-level operations. It is aimed at BYOVD and anti-cheat researchers studying mhyprot2's vulnerable interface.
