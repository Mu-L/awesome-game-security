This project is an IDA Pro plugin that integrates YARA rule scanning into the disassembly workflow. It allows running YARA rules against the loaded binary directly from within IDA, highlighting matches in the disassembly view and providing navigation to matched locations. The plugin supports custom rule sets for identifying malware signatures, crypto constants, or packer patterns. It is aimed at malware analysts and reverse engineers using YARA-based detection within IDA Pro.
