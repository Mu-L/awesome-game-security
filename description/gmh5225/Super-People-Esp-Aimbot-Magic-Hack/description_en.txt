This project is a cheat for Super People (battle royale game) providing ESP, aimbot, and gameplay modification features. It reads game entity data from the Unreal Engine-based process to display player positions, health, and distance, with aimbot targeting and other combat advantages. The C++ cheat interacts with the game's UE4 SDK structures. It is aimed at game security researchers studying UE4-based battle royale cheat implementations.
