This project is an AI-powered FPS aimbot that uses computer vision and machine learning to detect and aim at enemy players in first-person shooter games. It captures game frames, runs object detection (typically YOLO-based) to identify player models on screen, calculates target coordinates, and moves the mouse to aim at detected targets. The Python/C++ tool operates externally through screen capture. It is aimed at game security researchers studying AI/ML-based cheat detection and computer vision aimbot architectures.
