This project is an automation bot for Honkai: Star Rail's Simulated Universe game mode. It automates the repetitive gameplay loop of navigating the Simulated Universe dungeon, selecting blessings, fighting battles, and collecting rewards using screen recognition, automated input, and decision-making logic. The tool handles pathfinding, combat rotation, and blessing selection strategies. It is aimed at Honkai: Star Rail players automating farming content and game automation researchers.
