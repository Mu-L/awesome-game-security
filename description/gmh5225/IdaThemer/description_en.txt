This project is a tool for converting Visual Studio Code (VsCode) themes into compatible themes for IDA Pro.
This utility leverages a modified version of the Long Night project with Celestial as a basetheme and then remaps the colors defined in the VsCode theme JSON file to create a visually appealing and functional theme for IDA Pro.
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / ida themes area.
