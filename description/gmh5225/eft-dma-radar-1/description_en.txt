This project is a DMA-based radar cheat for Escape from Tarkov that reads game memory through PCILeech-compatible DMA hardware. It extracts player positions, loot locations, and map data from EFT's Unity game engine memory and displays them on a separate screen as a real-time radar overlay. The DMA approach makes the cheat invisible to BattlEye's process-level detection. It is aimed at game security researchers studying DMA-based cheat architectures for BattlEye-protected Unity games.
