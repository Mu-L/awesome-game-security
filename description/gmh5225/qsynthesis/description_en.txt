This project is QSynthesis, an IDA Pro plugin for synthesizing simplified expressions from complex obfuscated code using program synthesis techniques. It applies oracle-guided synthesis and SMT solving to find simpler equivalent expressions for MBA (Mixed Boolean-Arithmetic) obfuscated computations found in protected binaries. The Python plugin integrates with IDA's decompiler output. It is aimed at reverse engineers deobfuscating VM-protected or MBA-obfuscated code within IDA Pro.
