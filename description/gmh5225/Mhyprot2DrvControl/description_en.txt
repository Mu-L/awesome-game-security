This project is a control library for miHoYo's mhyprot2.sys anti-cheat driver, providing a convenient API for exploiting its vulnerable IOCTL interface. It wraps mhyprot2's kernel operations including process memory read/write, module enumeration, and process termination into a user-friendly C++ interface for BYOVD exploitation. It is aimed at BYOVD researchers using mhyprot2 as a kernel access primitive.
