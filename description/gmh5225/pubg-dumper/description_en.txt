This project is a PUBG (PlayerUnknown's Battlegrounds) SDK and offset dumper that extracts class definitions, property offsets, and function addresses from the running PUBG game process. It scans PUBG's Unreal Engine structures to dump GObject/GNames arrays, entity class hierarchies, and game-specific data layouts for cheat development. It is aimed at game security researchers studying PUBG's UE4 memory layout and anti-cheat researchers analyzing cheat tooling.
