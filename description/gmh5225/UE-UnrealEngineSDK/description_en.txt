This project is a reverse-engineered Unreal Engine SDK providing C++ class definitions, struct layouts, and function signatures for UE4/UE5 game internals. It includes UObject hierarchy definitions, UWorld/ULevel structures, player controller classes, actor component layouts, and engine globals needed for game hacking and modding. The SDK enables direct interaction with UE game objects from injected code. It is aimed at UE game hackers and modders who need SDK headers for internal cheat or mod development.
