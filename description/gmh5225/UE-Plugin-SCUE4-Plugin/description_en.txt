This project is SCUE4, an Unreal Engine 4 plugin that adds anti-cheat and security features to UE4 games. It provides runtime integrity checks, memory protection for sensitive game variables, encryption for network packets, and tamper detection for game binaries. The C++ plugin integrates into UE4's build system as a game module. It is aimed at UE4 game developers adding anti-cheat protection and game security researchers studying engine-level security plugins.
