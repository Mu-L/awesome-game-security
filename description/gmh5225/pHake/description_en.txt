This project is pHake, a GTA V mod menu and cheat framework providing vehicle spawning, teleportation, money modifications, player ESP, god mode, and other gameplay alterations. It hooks into GTA V's RAGE engine using ScriptHookV or direct memory access to manipulate game state. The C++ codebase demonstrates open-world game cheat architecture. It is aimed at game security researchers studying GTA V modding and anti-cheat evasion in Rockstar's titles.
