This project is the Turbulenz Engine, an open-source HTML5 game engine written in TypeScript/JavaScript. It provides a full game development framework with 3D rendering (WebGL), physics, audio, networking, input handling, asset management, and a scene graph. The engine includes both client-side game runtime and server-side services for leaderboards, multiplayer, and metrics. It is aimed at web game developers building browser-based 2D and 3D games using web technologies.
