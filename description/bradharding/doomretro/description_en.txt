This project is DOOM Retro, a classic DOOM source port for Windows written in C using SDL2. It enhances the original DOOM engine with widescreen rendering, uncapped framerate, improved lighting, texture filtering, gamepad support, and numerous bug fixes while preserving authentic gameplay feel. The port supports all original WAD formats, DeHackEd patches, and BOOM-compatible map features. It is aimed at classic FPS enthusiasts, DOOM modders, and game engine researchers studying idTech 1 engine modernization.
