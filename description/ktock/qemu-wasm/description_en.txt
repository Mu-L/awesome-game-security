This project is a WebAssembly port of QEMU that runs virtual machines directly in web browsers. It compiles QEMU's x86 and other architecture emulators to WASM using Emscripten, enabling users to boot Linux, Windows, and other operating systems in a browser tab without server-side processing. The project handles VirtIO device emulation, networking, and storage through browser APIs. It is aimed at emulation researchers, educators, and developers wanting browser-based virtual machine access.
