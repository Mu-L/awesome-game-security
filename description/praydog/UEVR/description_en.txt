This project is UEVR, a universal Unreal Engine VR injection mod that adds 6DOF VR support to virtually any Unreal Engine 4/5 game. It hooks into the engine's rendering pipeline to implement stereoscopic rendering, head tracking through OpenXR/OpenVR, motion controller input, and roomscale movement without game-specific modifications. The C++ injector uses UE reflection and pattern scanning to locate engine internals at runtime. It is aimed at VR enthusiasts and UE researchers adding VR support to flat-screen UE games.
