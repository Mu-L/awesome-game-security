LplPlugin (FullDive Engine) is a modular, experimental ultra-optimized C++23 engine for neuro-immersive game and simulation workloads. It is organized as many independent static libraries built with xmake, covering a data-oriented ECS, DAG task scheduling, deterministic fixed-point math, Morton spatial partitioning, Vulkan rendering, and optional CUDA compute. Networking uses lockless zero-copy IPC via a Linux kernel module with ring buffers for low-latency I/O, alongside UDP transport and anti-tunneling support. A dedicated BCI stack integrates OpenBCI and related sources with real-time DSP (FFT, Riemannian and Mahalanobis metrics) for neural input. Primary use cases are research and prototyping of high-performance game clients/servers, immersive simulations, and brain–computer interface driven gameplay on Linux.
