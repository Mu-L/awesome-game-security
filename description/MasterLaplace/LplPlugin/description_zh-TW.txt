LplPlugin（FullDive Engine）是一款模組化、實驗性的超優化 C++23 引擎，面向神經沉浸式遊戲與模擬負載。它由多個獨立靜態函式庫組成，使用 xmake 建置，涵蓋面向資料的 ECS、DAG 任務排程、確定性定點數學、Morton 空間劃分、Vulkan 渲染以及可選的 CUDA 運算。網路層透過 Linux 核心模組與環形緩衝區實作無鎖零拷貝 IPC，以支援低延遲 I/O，並搭配 UDP 傳輸與反隧道能力。專用 BCI 堆疊整合 OpenBCI 及相關資料來源，並提供即時 DSP（FFT、黎曼與馬氏度量）以處理神經輸入。主要用途是在 Linux 上研究與原型開發高效能遊戲用戶端/伺服器、沉浸式模擬，以及腦機介面驅動的玩法。
