This project is QtScrcpy, a Qt-based GUI application for displaying and controlling Android devices from a PC over USB or TCP/IP. It uses scrcpy's server component to capture the device screen via MediaCodec and streams it to the desktop with low latency, supporting keyboard/mouse input, file drag-and-drop, screen recording, and multi-device management. The C++ application works without requiring root access on the Android device. It is aimed at Android developers, testers, and users wanting to mirror and control Android devices from desktop.
