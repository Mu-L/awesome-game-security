This project is a card game implementation built with Unreal Engine 5. It demonstrates UE5 game development patterns including card deck management, turn-based gameplay logic, UI widget design with UMG, network replication for multiplayer, and visual effects for card animations. The Blueprint and C++ project serves as a learning reference for UE5 game development. It is aimed at Unreal Engine developers studying card/board game implementation patterns in UE5.
