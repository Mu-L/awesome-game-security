This project is MapleEzorsia, a MapleStory v83 private server source with custom content and modifications. It implements server-side game logic for the classic v83 version including login handling, character management, map instances, monster mechanics, quest systems, and custom gameplay features. The Java codebase extends the standard v83 server framework with quality-of-life improvements. It is aimed at MapleStory private server developers and game server researchers.
