This project is an external IL2CPP runtime resolver for Unity games that reads IL2CPP metadata from outside the game process. It parses the global-metadata.dat file and IL2CPP binary structures through external memory reading to resolve class names, method addresses, field offsets, and type information without injection. The C++ library enables external cheat development for IL2CPP Unity games. It is aimed at game hackers building external cheats for IL2CPP-based Unity titles.
