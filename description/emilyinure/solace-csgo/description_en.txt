This project is Solace, a CS:GO internal cheat written in modern C++ with a polished ImGui menu interface. It provides ESP, aimbot, triggerbot, movement assistance (bunny hop, auto-strafe), skin changer, and visual modifications through Source engine SDK hooking. The clean codebase demonstrates well-structured internal cheat architecture with modular feature organization. It is aimed at game security researchers studying CS:GO cheat design patterns and anti-cheat detection surfaces.
