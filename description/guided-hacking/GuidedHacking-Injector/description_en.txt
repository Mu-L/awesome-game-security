This project is a feature-rich Windows DLL injector with a C++ GUI built on Qt. It supports multiple injection methods including LoadLibrary, manual mapping, thread hijacking, NtCreateThreadEx, QueueUserAPC, and kernel-mode injection via a vulnerable driver. The tool includes options for DLL cloaking (PEB unlinking, header erasure), import resolution, TLS callback execution, and exception handler registration. It is aimed at game hacking learners and security researchers studying DLL injection techniques and their detection surface.
