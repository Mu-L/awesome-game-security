This project is Xidi, a Windows DLL that intercepts XInput and DirectInput API calls to provide virtual controller mapping and input translation. It acts as a compatibility layer allowing modern XInput controllers to work with legacy DirectInput games and vice versa, with configurable button mapping, deadzone adjustment, and force feedback translation. The C++ library loads as a proxy DLL. It is aimed at game players and developers solving controller compatibility issues between DirectInput and XInput APIs.
