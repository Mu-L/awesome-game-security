This project is SE-tools, a collection of Nintendo Switch homebrew tools for game memory editing and cheat development. It includes a memory scanner, pointer searcher, cheat code manager, and real-time memory viewer that operate on the Switch through the Atmosphere CFW's dmnt:cht service. The tools support creating and applying cheat codes in the standard Atmosphere cheat format. It is aimed at Switch homebrew developers and game modders creating memory-based cheats for Nintendo Switch titles.
