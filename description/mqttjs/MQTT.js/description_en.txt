Full-featured MQTT client library for Node.js and browsers supporting MQTT v3.1, v3.1.1, and v5.0 protocols over TCP, TLS, WebSocket, and WSS transports. Provides publish/subscribe messaging with QoS 0/1/2, automatic reconnection, last-will-and-testament, and CLI tools (mqtt pub/sub), with Electron and Vite integration examples.
