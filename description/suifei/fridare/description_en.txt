This project is a Frida repackaging and anti-detection tool for iOS and Android platforms. It provides shell scripts and a Go-based hex replacement utility that patches recognizable Frida strings, symbols, and artifact names to evade common anti-instrumentation detection, along with automated installation scripts and documentation for both rooted and rootless iOS setups. It is mainly useful for mobile reverse engineers who need stealth Frida deployments that bypass anti-Frida detection mechanisms.
