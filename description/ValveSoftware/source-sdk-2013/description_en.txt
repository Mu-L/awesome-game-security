This project is the Source SDK 2013 codebase for creating Source 1 game mods.
It includes large C++ game and engine code for titles such as Half-Life 2, Half-Life 2: Deathmatch, and Team Fortress 2.
The repository provides build pipelines for Visual Studio on Windows and Steam Runtime based builds on Linux.
It is intended for developers who want to build, customize, and distribute non-commercial Source engine mods.
