This project is a Windows tool for spoofing NVIDIA GPU hardware identifiers used for HWID (Hardware ID) banning in games. It modifies GPU serial numbers, device IDs, and other identifiers reported through NVIDIA's driver interface to evade hardware bans. The tool patches driver-level queries and registry entries that anti-cheat systems use for hardware fingerprinting. It is aimed at game security researchers studying HWID spoofing techniques and anti-cheat hardware identification methods.
