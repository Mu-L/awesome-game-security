This project provides a command-line tool for windows that automates adding drivers to the HVCI (HvciDisallowedImages) custom blocklist.
Hypervisor-Protected Code Integrity (HVCI) , often marketed as Memory Integrity in Windows Security settings, is a critical virtualization-based security (VBS) feature.
It is mainly useful for low-level Windows, Linux, and mobile researchers working in the some tricks / windows ring3 area.
