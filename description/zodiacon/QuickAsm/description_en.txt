This project is QuickAsm, a lightweight x86/x64 assembly editor and executor for Windows. It provides a simple GUI where users can type assembly instructions, assemble them in real time using the Keystone engine, view the resulting machine code bytes, and execute the assembled code within the application's process. The C++ tool is useful for quick assembly experimentation and shellcode testing. It is aimed at reverse engineers, exploit developers, and security researchers who need to rapidly prototype and test assembly code snippets.
