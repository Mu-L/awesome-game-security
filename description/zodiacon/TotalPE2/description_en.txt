This project is Total PE 2, a comprehensive PE (Portable Executable) file viewer for Windows with a modern WPF interface. It displays detailed PE structure information including headers, sections, imports, exports, resources, debug directories, relocations, TLS, exception handling, and .NET metadata. The C# application supports drag-and-drop, hex viewing, and resource extraction. It is aimed at reverse engineers, malware analysts, and Windows developers who need a quick visual PE structure inspector.
