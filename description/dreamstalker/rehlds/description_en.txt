A reverse-engineered and bugfixed reimplementation of the Half-Life Dedicated Server (HLDS build 6152/6153) created using DWARF debug info from the original Linux engine_i486.so binary.
It fixes numerous defects found during reverse engineering while maintaining full compatibility with the original GoldSource engine protocol, HLTV, and game mods.
It is mainly useful for game server operators, engine researchers, and modders studying GoldSource engine internals and running improved Half-Life 1 dedicated servers.
